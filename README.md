# Sticky Roller

Roll a sticky ball down a street — everything you touch sticks to you.  
Small = fast + precise, medium = wobbly, huge = slow + destructive.

## Run

1. Install deps:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

## Controls

- **WASD / Arrow keys**: steer + accelerate
- **R**: reset (new randomized street)
